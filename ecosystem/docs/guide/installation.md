# 安装

## 包管理器

### npm

```bash
npm install @xto/base @xto/form @xto/data
```

### pnpm

```bash
pnpm add @xto/base @xto/form @xto/data
```

### yarn

```bash
yarn add @xto/base @xto/form @xto/data
```

## CDN

```html
<script src="https://unpkg.com/@xto/base"></script>
```
