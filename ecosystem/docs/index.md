# Xto UI

一个基于 Vue 3 的组件库，拥有优雅的设计和强大的功能。

## 核心理念

Xto，代表 简约、高效与卓越，这正是 xto 的核心精神：

⚡ **极速体验**

xto 追求极致性能，组件轻量化设计，按需加载无冗余。每个组件都经过精心优化，让你的应用如闪电般响应。

🎯 **简洁优雅**

代码如诗，界面如画。xto 提供直观的 API 设计和一致的交互体验，让开发者专注于业务逻辑，创造简洁而强大的用户界面。

🚀 **持续进化**

技术永不止步。xto 紧跟 Vue 3 生态发展，通过模块化架构和灵活的 API 设计，不断迭代升级，确保项目始终处于最佳实践前沿。

🔧 **灵活可控**

高度可定制的主题系统，从颜色到组件样式，一切尽在掌控。xto 让你在标准化与个性化之间找到完美平衡。

## 特性

- 🎨 **优雅设计** - 美观独特的 UI 组件
- 🧩 **模块化** - 支持按需加载，Tree-shakable
- 🌍 **TypeScript** - 完整的 TypeScript 支持
- 🎯 **Composition API** - 基于 Vue 3 组合式 API 构建
- 📱 **响应式** - 适配所有设备
- 🌙 **主题定制** - 强大的主题自定义能力

## 快速开始

```bash
# 安装
pnpm add @xto/base @xto/form @xto/data

# 使用
import { Button } from '@xto/base'

<Button type="primary">点击我</Button>
```
